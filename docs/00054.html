Hardware Setup
  1. Project sam_rh71_xult.X
    • Hardware Used
      • SAM RH71 Evaluation Kit
      • LX7720 board
      • Hurst DMA0204024B101 BLDC Motor
    • Hardware Setup
      • Connect the debugger probe to J33
      • Connect the USB port on the board to the computer using a mini USB cable.
      • Connect the Hurst DMA0204024B101 BLDC motor's phase windings and encoder connections as specified in the table below
