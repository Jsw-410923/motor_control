Introduction

This library component provides Field Oriented Control (FOC) algorithm for Permanent Magnet Synchronous Motor (PMSM). 

 

PMSM_FOC connects to the Peripheral Libraries (PLIBs) from CSP repository and X2CScope from X2C repository. 

This component needs to be connected to ADC, PWM PLIBs for correct functioning. PLIB name can be different in different devices. 

If position feedback is quadrature encoder sensor, then it depends upon QDEC peripheral. 

 

This component has support for following devices.

 
MCLV2 
MCHV3 
SAM E70 
Yes 
No 
PIC32MK MCF 
Yes 
No 
PIC32MK MCM 
Yes 
No 

 

PMSM_FOC component supports following features.

Feature 
Options 
Position Feedback 

  • PLL estimator
  • Quadrature encoder

 

Current measurement 

  • Dual shunt

 

Control loop 

  • Speed mode
  • Torque mode
  • Open loop (for PLL estimator)

 

Field Alignment 

  • Q-axis alignment
  • D-axis alignment

 

Field weakening 

  • Equation based field weakening
